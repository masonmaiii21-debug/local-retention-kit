# Outreach Scripts

Use these as templates, but customize the first sentence for every shop. Mention something real: a review, service, Instagram post, or missed owner reply.

## Instagram / Facebook DM

```text
Hi [Name], I noticed [Business] has strong reviews, but a few recent ones do not have owner replies yet.

I build a small review + customer follow-up kit for pet grooming shops. It helps reply to reviews faster and remind past customers to rebook.

I can send 3 free reply/follow-up examples using your shop's tone if useful.
```

## Email

```text
Subject: Quick idea for [Business]

Hi [Name],

I noticed [specific observation about reviews, rebooking, or social page].

I help pet grooming shops set up a simple review reply and customer rebooking kit. It is not a complicated CRM. The goal is to make it easier to follow up with past customers, reply to reviews, and avoid losing people who asked about grooming but never booked.

I can send 3 free example messages based on your shop's tone if you want to see what it looks like.

Best,
Mason
```

## Follow-Up 1

```text
Hi [Name], quick follow-up. I wrote one sample reply using your shop's style:

[Insert one custom review reply]

If helpful, I can build a small 7-day kit with review replies, rebooking reminders, and lead follow-up messages.
```

## Follow-Up 2

```text
No worries if now is not the right time. I am testing this with a few local grooming shops and keeping the first setups simple: no subscription, just a reusable message kit and tracking sheet.
```

## Free Sample Format

Send 3 samples:

1. One Google review reply
2. One rebooking reminder
3. One unanswered inquiry follow-up

Keep the sample short. The point is to start a conversation, not deliver the whole package for free.
